# Aarflingo branding

| File | Use |
|------|-----|
| `Aarflingo-logo.png` | **Canonical** product logo (studio header, Electron icon, README) |
| `logo.png` | Legacy Deepiri mark (optional) |

Sync into the studio app:

```bash
./scripts/sync-branding.sh
```

`./setup.sh` runs this automatically before `npm install`.
